本網站為 GitHub Pages 上的靜態網站。只有公開來源或經幹部核准的內容會進入網站。

## 資料來源

| 內容 | 來源 | 更新方式 |
|------|------|----------|
| 首頁竹韻近期動態 | 臺灣口琴觀測站竹韻專用公開 API | 瀏覽器直接讀取最小必要欄位，不需重新部署；失敗時保留內建備援資料 |
| 首頁完整活動時程 | 公開 Google Calendar | 幹部編輯行事曆，首頁直接顯示 |
| 幹部名單 | 公開 Google Sheet | 只顯示核准公開的職稱與姓名 |
| 相簿 | repo 內核准照片 + 公開 Google Sheet 索引 | 幹部提供照片，由網站管理員整理上傳 |
| 社群連結 | 公開 Google Sheet | 只放官方公開入口 |

## 更新頻率

GitHub Actions 每日同步公開 Google Sheet 四次；有內容變更時才提交快照並重新部署。首頁的竹韻公開社群動態由瀏覽器直接讀取觀測站專用 API，不會因新貼文建立官網 commit。完整活動時程直接來自公開 Google Calendar。

## 資料快照

公開 CSV 與觀測站近期動態的 last-good 快照隨網站原始碼保存，可從版本紀錄回復。觀測站失效、回傳不合法或沒有符合條件的資料時，前端不會替換 Hugo 輸出的備援卡片。觀測站快照只由網站管理員手動更新；頁尾會區分「Repo 資料快照」與「Google Sheet 資料」，時間不代表內部資料或 Google Sheet 的最後編輯時間。

## 問題回報

若發現網站內容錯誤或連結失效，請透過[關於我們](/about/)頁面的聯絡方式告知幹部。
